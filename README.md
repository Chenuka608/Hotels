# LuxeStay ✨

A luxury hotel showcase website for Sri Lankan resorts and hotels, built with **Next.js 15** and **Tailwind CSS**.

## 🌟 Features

- ✅ Fully responsive design (mobile-first)

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Chenuka608/Hotels.git
cd hotel

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
